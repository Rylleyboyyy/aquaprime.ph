INSTALLATION STEPS:

1. Download and Install XAMPP
2. Open Xampp control panel and start APACHE and MYSQL
3. Navigate Xampp Installation folder (usually under C:\xampp\htdocs)
4. Open Browser and go to (localhost/phpmyadmin)
5. Create a new database named aquaprime_db
6. Click the Import tab and upload the aquaprime_db.sql
7. Enjoy the website by going to (localhost/aquaprime.ph)
